Background music files used by game states:
- Start.ogg (start screen loop)
- Wave.ogg (enemy waves loop)
- Boss Fight.ogg (boss fight loop)
- End.ogg (ending, no loop)


## Path Lock
Use these exact paths in code:
- `./public/assets/audio/Start.ogg`
- `./public/assets/audio/Wave.ogg`
- `./public/assets/audio/Boss Fight.ogg`
- `./public/assets/audio/End.ogg`

Do not rename or change these paths unless explicitly requested by the user.
